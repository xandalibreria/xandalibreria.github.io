Inatalación

Para implementar XandA mediante el método de instalación, necesitas usar el gestor de paquetes npm, una vez cuentes con esta herramienta abre la terminal o el CMD de tu computadora, dirígete a la carpeta de tu proyecto e inserta esta línea de código:

npm install XandA

Al hacerlo se te generará una carpeta en la raíz de tu proyecto con el nombre "" en la cual estarán albergados todos los recursos de la librería XandA, sin embargo todavía necesitas cargar los archivos pertinentes en tu HTML o PHP, tanto dentro del head como dentro del body, tal y como se muestra en el siguiente ejemplo:

<!DOCTYPE html>
<html lang="es">
<head>
<title>index</title>

<!-- Resto de tu código -->

<!-- XandA -->
<link rel="stylesheet" href="../../assets/styles/documentacion.css">
</head>

<body>

<!-- Resto de tu código -->

<!-- XandA -->
<script href="../../..."><script>
</body>
</html>

Actualizaciones

Para actualizar la librería deberás dirigirte a la terminal o CMD de tu computadora, allí navega a la carpeta de tu proyecto y escribe el siguiente comando:

npm update XandA
